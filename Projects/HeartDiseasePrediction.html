Heart Disease Prediction

Purpose

The purpose of this project is to predict that whether the patient has heart disease or not

Input and output

Input: Person

Output: Prediction (Have Disease / Not Have Disease)

Representation of Input (Person)

Input (Person) is Represented as 4 Input Attributes

Input Attributes 01- Age

Possible Value: Any positive integer between 1 to 100

Input Attributes 02 - Gender

Possible Values

  • Male
  • Female

Input Attributes 03 - Chest Pain

Possible Values

  • 0
  • 1
  • 2
  • 3

Input Attributes 04 - Chest Pain

Possible Value: Any positive integer between 0 to 220

Representation of Output (Disease)

Output is Represented as a single Output Attributes

  • Have Disease
  • Not Have Disease
Example - Input and Output
Input (Person):

Age:

58

Gender:

Female

Chest Pain:

0

Maximum Heart Rate Achieved:

122

Output: Have Disease

Person Input

Age: sdaf

Gender: asd

Chest Pain: fdf
